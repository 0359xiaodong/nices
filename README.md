NiceS一个简单的新闻app

说明
-------------------------------------------------------------------------------------------------------------------------------------------------------
NiceS只是作者无聊的时候，为了不荒废时间，就在eclipse上随笔画画，网上超超做出的小东西。功能，就是看看文章，收藏文章而已，呵呵，真的
好简单，嗯，所以代码也挺简单的。用到的东西，什么toolbar,涟漪效果，5.0刷新控件，仿qq删除，自定义toast,自定义progress....我也不记得了
